# mofish 

get site hot list. (Rust)

- [x] HackerNews
- [x] Hupu
- [x] Douban

## Usage

```
# generate `mofish.json`
cargo run
```

you can use other tools to read `mofish.json`

or 
```
sh render_mofish.sh
```
